# Start Here

This section establishes how to use the pack without confusing companion guidance with authoritative requirements.

## Complete this section when you can

- identify the authority source for a decision;
- name the party authorised to make the local decision;
- distinguish interpretation from implementation choice;
- identify the evidence that proves the decision was enforced; and
- identify the trigger that requires reassessment.
