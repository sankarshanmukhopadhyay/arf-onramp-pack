## Important corrections in this release

- **Regulation (EU) 2024/1183** is an amending regulation, not a Commission implementing regulation
- technical-specification tracking belongs in the STS repository
- credential-specific rulebook tracking belongs in the attestation rulebooks catalog
- the on-ramp should be treated as implementation guidance, not as the source of binding obligations
