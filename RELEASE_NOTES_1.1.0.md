# Release Notes 1.1.0

## Summary

This release prepares the ARF On-Ramp Pack for a stronger maintenance posture and a more accurate authority model.

## Highlights

- added governance synchronization control plane
- added upstream drift monitoring and issue automation
- completed an April 2026 semantic re-audit of the companion documentation
- corrected treatment of Regulation (EU) 2024/1183
- aligned repository references to the canonical ARF and STS upstream repositories
- refreshed implementation and reading-path guidance

## Why this release matters

The repository now has a clearer distinction between:

- law
- implementing acts
- upstream technical guidance
- local companion interpretation

It also has a machine-verifiable maintenance mechanism for upstream drift.
