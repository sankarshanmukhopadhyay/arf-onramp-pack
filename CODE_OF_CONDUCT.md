# Code of Conduct

This project supports professional, evidence-led collaboration around EUDI Wallet implementation guidance.

Contributors are expected to keep disagreement focused on sources, authority boundaries, implementation impact, and audit evidence. Harassment, personal attacks, sustained disruption, or exclusionary conduct are not acceptable.

Maintainers may moderate comments, close issues, or restrict participation when behavior prevents constructive work.

Report concerns through the repository maintainers' normal private contact channel.
