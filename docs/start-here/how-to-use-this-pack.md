# How to Use This Pack

Each guided page uses a common operating pattern:

1. **Learning objective** establishes the required capability.
2. **Authoritative inputs** identify what must be consulted.
3. **Decision checkpoint** requires an explicit governance or implementation choice.
4. **Produce this artifact** turns reading into evidence.
5. **Validation criteria** make completion testable.
6. **Previous / Next** maintains a deterministic sequence.

{: .evidence }
> A journey is complete only when its named artifact exists, has an accountable owner, and can be reviewed against authoritative inputs.
