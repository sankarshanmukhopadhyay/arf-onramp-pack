# Evidence Planning

Plan evidence at the same time as the control, not after implementation.

## Evidence classes

- approved policy or architecture decision;
- configuration and deployment record;
- conformance or interoperability test result;
- security and privacy assessment;
- operational telemetry and incident record;
- change, revocation, and supersession record.

## Validation criteria

Evidence must be attributable, time-bounded, integrity-protected, reproducible where applicable, and linked to the authoritative source and local decision it supports.
